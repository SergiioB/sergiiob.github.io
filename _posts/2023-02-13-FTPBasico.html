Instalacion de FTP

1-Instalación

sudo apt-get install vsftpd

Iniciar el servicio

sudo systemctl start vsftpd

Comprobar el estado del servicio

sudo systemctl status vsftpd
sudo systemctl status vsftpd

2-Configuracion

  • Tipo de servidor: Elegir Sitio de internet.El correo se envia y se recibe directamente utilizando SMTP
  • Nombre del sistema de correo: Pongamos que el nombre del servidor es mail.ejemplo.com. Seguramente querrás establecer como nombre del sistema de correo ejemplo.com, para que, para el usuario user1, Postfix va a usar la dirección user1@ejemplo.com
  • Recipiente de correo para el administrador y postmaster Esta es la cuenta de linux a la que se le van a reeenviar los correos enviados a root@ y postmaster@. Usa tu cuenta primaria para ello, en este caso, serge.
  • Otros destinos para aceptar correo: Esto define los destinos a los que Postfix va a aceptar. Si necesitas añadir cualquier otro dominio en este servidor al que el servidor se va a encargar de recibir, añadelos aqui. Si no, por defecto es suficiente.
  • Forzar actualizaciones síncronasSeguramente estes usando un sistema de sarchivos con jorunal, asi que, pulsa NO
